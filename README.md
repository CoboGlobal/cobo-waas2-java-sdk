### CoboAuth

- **Type**: API key
- **API key parameter name**: BIZ-API-KEY
- **Location**: HTTP header

<a id="CoboSignature"></a>
### CoboSignature

- **Type**: API key
- **API key parameter name**: BIZ-API-SIGNATURE
- **Location**: HTTP header

<a id="CoboNonce"></a>
### CoboNonce

- **Type**: API key
- **API key parameter name**: BIZ-API-NONCE
- **Location**: HTTP header
